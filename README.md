# 📈 MarketPulse AI: 5-Minute Forecaster

A real-time financial dashboard that uses Machine Learning (Random Forest) to predict market movements in 5-minute intervals.

## 🚀 Features
- **Real-time Ingestion:** Fetches live data via Yahoo Finance API.
- **ML Brain:** Random Forest Classifier trained on RSI, Volatility, and SMA trends.
- **Live Dashboard:** Built with Streamlit for sub-60s updates.

## 🛠️ Tech Stack
- **Language:** Python
- **Libraries:** Scikit-learn, Pandas, Streamlit, Plotly, YFinance

## 📖 How to Run
1. Clone the repo: `git clone <your-link>`
2. Install dependencies: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`
